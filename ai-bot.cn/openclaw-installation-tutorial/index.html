OpenClaw是什么

OpenClaw 是开源的 AI 代理平台，核心理念是将 AI 从“被动回答问题”转变为“主动执行任务”。传统的 AI 工具如 ChatGPT、Claude 等，主要是通过输入提示来获得被动响应，而 OpenClaw 的目标是创建一个可以主动执行任务的数字员工。能通过聊天应用（如 WhatsApp、Telegram、Discord 等）接收指令，自动完成日常任务，如邮件管理、日程安排、数据抓取等。

  • 2025年11月：Peter Steinberger 发布 Clawdbot 基于其个人 AI 助手 Clawd（后改名 Molty）。

  • 2026年1月30日：最终更名为 OpenClaw，创始人承认 Moltbot，”说起来不顺口”，新名经过商标检索和域名保护。

这段历程印证了快速迭代在开源项目中的双刃剑效应：既能快速响应危机，也可能因过快而留下安全隐患。

OpenClaw与其他AI工具的区别

工作方式

  • OpenClaw：主动执行任务的数字员工，支持丰富的功能扩展，通过技能（Skills）平台可以添加和管理各种任务。支持第三方集成提供了 50+ 种第三方应用的集成，包括生产力套件、智能家居设备、流媒体平台等，打破了数字与物理世界的界限。
  • ChatGPT：被动回答的对话式 AI 工具，依赖用户的输入提示来生成回答。不能主动去执行用户未明确请求的任务，专注于生成连贯且富有信息的文本。
  • Claude：和 ChatGPT 类似，主要用于生成自然语言文本和回答问题。Claude 也能处理复杂的语言任务，但不具备 OpenClaw 任务自动化能力。重点在安全性与伦理，Claude 强调人工智能的安全性和伦理性，减少 AI 系统的偏见和潜在风险。

交互方式

  • OpenClaw：支持与用户在多种聊天应用中进行交互，如 WhatsApp、Telegram、Discord、Slack 等。OpenClaw 能记住上下文并在持续对话中维持记忆，在多次交互中提高效率，避免每次都从头开始。
  • ChatGPT 和 Claude：局限于对话输入：能在短时间内处理持续对话，不具备跨多次会话的持久性或自动化执行任务的能力。API 集成能力较弱，不能集成多个外部服务和工具进行任务自动化

部署与使用方式

  • OpenClaw：支持 本地部署，用户可以将 OpenClaw 部署在自己的本地计算机上。支持 云端部署，OpenClaw 可以部署到云服务器，适用于需要 24/7 全天候运行的场景。支持自定义 技能开发，OpenClaw 提供了一个技能市场（ClawHub），允许用户根据自己的需求定制和安装新的技能。
  • ChatGPT 和 Claude：在线服务，使用主要依赖于云端服务，用户需要通过网站或第三方平台进行访问，不支持本地部署。无技能市场，无法提供针对不同任务的深度定制。

安全性与隐私

  • OpenClaw：OpenClaw 的安全性需要特别关注，因为允许第三方集成，能访问用户的个人数据和应用程序。官方文档中提到了一些安全风险，如恶意技能、暴露实例和零点击攻击，为了降低风险，用户应当遵循安全最佳实践，如隔离环境部署、使用 Docker 沙盒、启用身份验证等。
  • ChatGPT 和 Claude：安全控制较为集中，ChatGPT 和 Claude 的安全性更多地由服务提供商（OpenAI 和 Anthropic）进行管控，用户无需直接管理部署中的安全问题，局限性在于无法定制和完全控制安全策略。

通过对比，根据实际需求选择最适合的工具。如果需要的是一个具有强大自动化和集成功能的 AI 代理，OpenClaw 是更合适的选择；如果只是进行文本生成和对话，则 ChatGPT 或 Claude 更为合适。

记忆能力 会话级 长期记忆+上下文
执行能力 只给建议 直接操作工具

OpenClaw的安装部署教程

包含本地部署，云端部署，接入飞书，钉钉的操作说明。

本地部署+Win版+接入飞书

准备工作

步骤1. 检查电脑是否符合要求：确认操作系统是macOS / Linux / Windows 10/11；内存大小要 ≥ 4GB（8GB 更流畅）；剩余硬盘空间 ≥ 10GB 可用空间。

步骤2. 安装Node.js

需要先安装Node.js，版本最好是最新的，或者必须 ≥ 22。

下载地址：https://nodejs.org/en/download/，根据电脑系统，选择下载并安装。

如果已经安装了，检查一下Node.js的版本

打开终端命令窗口，输入下面的命令：

node -v

这里可以看到具体的版本号是多少。

步骤3. 获取模型的API key

openclaw支持连多种第三方的大模型，如果你有Kimi，MiniMax，Claude，ChatGPT或Gemini等大模型的API，可以直接使用。

我们以智谱举例，访问智谱AI开放平台官网：https://bigmodel.cn/

选择API Key，点击添加新的API Key，复制API Key备用。

步骤4. 创建飞书机器人

如果你只是在本地电脑玩一下，可以不用操作这一步，可直接看下面安装OpenClaw。

这一步主要是为了方便你在手机上使用openclaw，随时接收或发送命令给openclaw。

打开飞书应用配置页面：https://open.feishu.cn/app

点击开发者后台，选择创建企业自建应用

输入应用名称，应用描述，选择一个头像，点击创建就行了

选择凭证与基础信息，这里可以看到机器人的Aapp id和APP 密钥（这两个要保密），下面配置openclaw需要用到。

安装OpenClaw

1.打开终端，打开 PowerShell（管理员）右键点击开始菜单，搜索”终端”，选择”以管理员身份运行”

输入以下命令，等待安装完成

npm install -g openclaw@latest

检查是否安装完成，输入下方命令

openclaw –version

能看到具体的版本号，表示安装成功。

2.配置安装向导与网关

这一步非常重要，配置一些openclaw的配置内容和网关，在终端输入命令

openclaw onboard –install-daemon

显示这个页面就对了

下面会进行安装指引，这里按键盘左方向←→键选择，第一个选择Yes

第二个配置方式，这里选默认的QuickStart

到配置大模型API，选择Z.AI

选择第二个国内使用的

进去后再粘贴你的API Key。

自己选择使用智普的哪个大模型。

配置聊天工具，选择飞书，如果只是网页体验，可以选择跳过，后面再需要也可以让OpenClaw帮忙配置。

按默认选项先安装飞书的配置文件，然后就会提示你输入飞书机器人的秘钥。

把提前准备好的秘钥和appid粘贴在这里即可。

然后选择群聊天策略，这里我们选运行群聊。

安装SKills

选择Yes，再选择跳过SKip for now（这里用空格选中），后续安装成功了也可以直接让openclaw安装这些Skills。

安装hooks

最后是安装hooks，这里可以理解成安装几个配置文件。建议都选择上，这里用空格键选中

到这就配置好了，等openclaw自动安装网关，生成网页链接，会自动打开网页页面。

3.匹配飞书配对码

打开飞书，找到开发小助手，打开应用，给机器人发送消息，比如“你好”，会弹出一个配对码。

在终端命令输入下面的命令

openclaw pairing approve feishu [你的配对码]

完成和飞书分配对。

4.设置飞书权限及配置

进入飞书应用后，选择权限管理，选择批量导入权限，在输入框中输入下面内容，然后点击右下角的【确认申请权限】-【确认开通】即可。

{
“scopes”: {
“tenant”: [
“contact:contact.base:readonly”,
“contact:user.base:readonly”,
“im:message”,
“im:message.group_at_msg:readonly”,
“im:message.group_msg”,
“im:message.p2p_msg:readonly”,
“im:message.reactions:read”,
“im:message:readonly”,
“im:message:recall”,
“im:message:send_as_bot”,
“im:message:update”,
“im:resource”
],
“user”: [
“docx:document:readonly”
]
}
}

5.事件订阅与回调

选择事件配置，点击订阅方式，选择推荐长链接方式。再点击添加事件，搜索 im.message.receive_v1 勾选功能，确认添加。

6.创建版本

点击创建版本，输入版本号，描述更新说明，点击保存，自动会审核发布。再次打开应用，就可以和机器人对话了，可以安排你的机器人执行任务。

云端部署+Maxclaw+接入钉钉

MaxClaw是Minimax推出的AI智能体创建平台，帮助用户快速构建和部署个性化的AI助手。可通过自然语言描述需求，快速创建具有特定技能的AI智能体，支持与飞书、微信、Telegram等即时通讯工具集成，AI助手可7×24小时在云端运行，主动执行定时任务并推送结果。数据存储在Minimax云端，提供一定量的云存储和长期记忆功能。

进入 MiniMax Agent  官网：https://agent.minimaxi.com

在左侧导航栏或者首页输入框上方选择 MaxClaw

点击立即开始，MaxClaw 会自动开始部署。

部署完成后，让MaxClaw帮我们接入钉钉。

先在钉钉开发者后台创建应用，添加机器人，获取 AppKey 和 AppSecret 。发给MaxClaw，自动进行配置。

部署完成后，在聊天中添加机器人，艾特机器人就可以交互，执行任务了。

OpenClaw其他一键部署方案

KimiClaw

Kimi 推出的官方 OpenClaw 托管服务，深度集成 Kimi K2.5 模型，无需额外配置 API Key，提供 40GB 免费存储空间，支持一键调用 ClawHub 社区插件。

安装部署步骤：
  • 访问 Kimi 官网，找到 Kimi Claw Beta 入口（目前仅对 Allegretto 及以上会员开放）。

  • 选择关联已有 OpenClaw 或创建新实例。

  • 如需接入飞书/钉钉，在「渠道配置」中选择对应平台并按引导完成授权。

Coze Claw

字节跳动扣子Coze推出的无代码部署方案，无需准备服务器或编写后端代码，通过可视化界面即可完成 OpenClaw 部署和飞书机器人接入。

安装部署步骤：
  • 登录扣子编程平台（code.coze.cn），确保账号为高阶版或旗舰版。
  • 首页点击「限时体验：一键部署 OpenClaw」或在「优秀案例」区找到 OpenClaw 项目。
  • 点击「创建副本」，填写应用名称和描述。
  • 系统自动完成部署，左侧对话框发送「你好」测试是否成功。
  • 接入飞书：登录飞书开放平台 → 创建企业自建应用 → 添加「机器人」能力 → 在 Coze 平台配置飞书 Bot 的 App ID 和 Secret → 完成事件订阅和权限配置。

百度云 Claw

百度智能云千帆大模型平台推出的 OpenClaw 快速体验功能，集成百度搜索、百科、学术检索、PPT 生成等官方 Skills，首次部署送 200 元代金券。

安装部署步骤：
  • 登录百度智能云千帆平台（qianfan.cloud.baidu.com）。
  • 进入「OpenClaw 一键体验」专题页面。
  • 点击「立即部署」，选择预置模型配置（内置文心大模型）。
  • 系统自动分配免费计算资源并完成部署（约 3-5 分钟）。
  • 部署成功后，在控制台点击「开始对话」测试基础功能。
  • 在「Skills 管理」中一键启用百度搜索、PPT 生成等官方插件。
  • 如需接入飞书/钉钉，在「渠道配置」中按引导完成 webhook 配置。

基于OpenClaw的衍生项目

NanoClaw

NanoClaw是轻量级的AI代理框架，核心代码仅约4000行，采用Linux容器隔离运行，提供可审计、安全的个人AI助手功能。通过技能（skills）实现功能扩展，例如输入/add-telegram即可自动修改代码集成Telegram，无需传统配置文件。

与OpenClaw相比，NanoClaw更注重极简设计，适合注重安全和易用性的用户，已被AI大牛Karpathy推荐。特色包括AI驱动的代码自我修改能力，以及支持智能体集群协作的新功能。

Nanobot

Nanobot是香港大学数据科学实验室（HKUDS）开源的超轻量级个人AI助手框架，提供简洁、高效且易于扩展的AI智能体解决方案。核心代码仅约4000行，相比同类项目（如OpenClaw的43万行代码）缩减99%，资源占用极低，启动速度快，适合在资源受限的设备上运行。

支持多平台接入（Telegram、Discord、WhatsApp、飞书等）、多模型适配（OpenAI、Claude、DeepSeek等），并具备记忆管理、工具调用、定时任务等核心功能，可满足日常办公、开发辅助、知识管理等场景需求。

采用MIT协议，适合个人开发者、科研人员及企业用于学习、实验或构建个性化AI助手。

PicoClaw

PicoClaw是超轻量级、高效能的个人AI助手，专为低资源环境设计。内存占用小于10MB，启动时间仅约1秒，可在10美元左右的RISC-V开发板等低成本硬件上运行，适合路由器、IP摄像头、树莓派等嵌入式设备。基于Go语言开发，支持RISC-V、ARM、x86等架构，提供Linux、Windows等系统的单文件可执行程序，无需复杂依赖，可快速部署。

支持多渠道聊天（如Telegram、Discord、飞书等），具备网络搜索、代码生成、文件操作、定时任务等能力，可实现自动化办公、智能家居控制等场景。

采用MIT协议开源，可离线部署，支持多种AI模型，适合智能家居控制、IoT传感器监控、离线边缘计算、嵌入式AI使用。

ZeroClaw

ZeroClaw是基于Rust语言开发的超轻量级AI助手框架，为生产环境打造的”安全优先”AI Agent，用架构设计解决OpenClaw的安全隐患。提供高性能、低资源占用且安全可靠的AI助手解决方案。可在树莓派、低配云主机等资源受限的设备上流畅运行。采用模块化设计，核心组件（如模型提供商、通信渠道、记忆系统等）均通过Trait接口实现插件化，用户可根据需求灵活替换或扩展功能，支持超过22种模型提供商（如OpenAI、Anthropic、DeepSeek等）。

内置配对机制、沙箱隔离、文件系统权限控制等安全特性，默认绑定本地地址，拒绝公网直接访问，有效防止数据泄露和未授权操作。完全自研记忆系统，无需依赖外部数据库（如Pinecone、Elasticsearch），通过SQLite实现向量搜索和关键词检索，确保数据自主性和隐私性。

ZeroClaw适合对性能、资源占用和安全性有严格要求的场景，如边缘设备部署、低成本云服务或需要高度定制化的AI助手开发。

推荐对数据隐私要求极高的个人用户；需要完全本地部署的企业环境；希望通过即时通讯工具远程控制电脑的用户；需要自动化处理本地文件和系统操作的专业人士等场景使用。

OpenClaw Skills常用推荐

一、基础必备技能

  1. Agent Browser
    功能：网页自动化操作，如打开网页、抓取信息、填表单、截图等。
    适用场景：几乎所有需要与网页交互的任务，如对比产品价格、收集网页数据等。
    安装命令：clawhub install agent-browser

  2. Brave Search
    功能：联网搜索，提供实时信息检索能力。
    适用场景：查询新闻、资料、答案等，摆脱AI知识截止日期限制。
    安装命令：clawhub install brave-search
    配置：需申请Brave Search API Key（免费额度可用）。

  3. Summarize
    功能：一键总结PDF、长文、视频、音频等内容，提取核心要点。
    适用场景：快速处理学术论文、工作报告、学习资料等。
    安装命令：clawhub install summarize

  4. Shell
    功能：执行终端命令，实现文件操作、脚本运行等系统级任务。
    适用场景：批量处理文件、自动化系统管理任务。
    安装命令：clawhub install shell
    注意：需谨慎使用，避免误操作。

  5. Cron/Wake
    功能：定时任务和提醒，可设置定期执行任务或发送提醒。
    适用场景：每日天气播报、周报提醒、定期检查邮件等。
    安装命令：clawhub install cron

二、生产力与知识管理

  1. Notion
    功能：与Notion API集成，操作页面、数据库，自动整理笔记和项目数据。
    适用场景：知识管理、项目跟踪、团队协作。
    安装命令：clawhub install notion

  2. Obsidian
    功能：读写本地Markdown笔记，支持标签管理、知识关联。
    适用场景：个人知识库构建、笔记整理。
    安装命令：clawhub install obsidian

  3. Gmail
    功能：自动处理邮件，如读取、回复、归档、搜索等。
    适用场景：邮件管理、自动回复常见问题。
    安装命令：clawhub install gmail
    配置：需OAuth授权。

  4. Google Calendar
    功能：日程管理，设提醒、排会议、推送通知。
    适用场景：会议安排、日程跟踪。
    安装命令：clawhub install google-calendar

三、开发与代码管理

  1. GitHub
    功能：管理GitHub仓库，创建Issue、查看PR、分析代码。
    适用场景：开发者日常协作、项目维护。
    安装命令：clawhub install github
    配置：需Personal Access Token。

  2. Frontend Design
    功能：生成生产级前端界面代码。
    适用场景：前端开发辅助、原型设计。
    安装命令：clawhub install frontend-design

四、安全与风险控制

  1. Skill Vetter
    功能：安装前扫描技能，检测潜在安全风险。
    适用场景：所有技能安装前必用，确保安全性。
    安装命令：clawhub install skill-vetter

  2. Link Checker
    功能：检测URL安全性和钓鱼链接。
    适用场景：处理外部链接时，防止安全威胁。
    安装命令：clawhub install link-checker

安装建议：

  • 先安装安全类技能（如Skill Vetter），再安装基础技能，最后按需添加其他技能。
  • 涉及敏感操作（如文件删除、账号授权）时，开启OpenClaw确认模式，避免误操作。
  • 定期更新技能：clawhub update --all，确保安全补丁和功能更新。
