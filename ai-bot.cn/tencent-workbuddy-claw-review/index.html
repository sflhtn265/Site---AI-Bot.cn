腾讯版”小龙虾” WorkBuddy 实测 – 接入微信远程派活，白嫖 5000 Credits

AI教程4个月前发布 AI小集
0 3

上周末，我在咖啡厅见朋友，朋友全程抱着电脑焦头烂额地改 Bug，咖啡凉了都没顾上喝。

那一刻我就在想，难道我们这代人打工，真的要被电脑锁死在座位上吗？

直到最近，我深度体验了腾讯 CodeBuddy 团队刚放出来的大招——WorkBuddy Claw（腾讯版小龙虾）。随时随地在手机上发条指令，AI 就把活干了。

这篇文章跟大家聊聊我这 2 天的真实使用感受，以及我是怎么“白嫖” Credits 和云服务器的。

官网：https://www.codebuddy.cn/work/

 

01. 远程遥控“龙虾”办公

 

以前我们用 AI 编程工具，流程通常是：打开电脑 -> 打开 IDE -> 输入需求 -> 等 AI 回复。如果出门在外或者躺在床上，灵感来了？对不起，等开电脑再说。

但 WorkBuddy 的逻辑变了。

WorkBuddy 长在了企业微信、微信客服、飞书、钉钉甚至 QQ 里。我们可以通过 Bot 联系人直接在手机上给它派活。

以微信为例：

点击【配置】之后，扫微信二维码就可以绑定上，比绑定飞书等渠道方便多了。

绑定好之后，客服消息里会出现 WorkBuddy 的打招呼信息。

点击【客服消息】就可以看到 WorkBuddy 了。

最近各个大厂都在推出自己的龙虾，几乎每天都有新龙虾跟大家见面。想要全面的了解是很消耗个人时间和精力的，我们可以直接交给 WorkBuddy：

在手机上发一条微信消息，WorkBuddy 电脑端就开始给我干活了。

2 分钟就完成了任务，效果非常不错了。这种图表，自己整理怎么不得几十分钟。

这就引出了 WorkBuddy 最核心的体验：本地+云端双模运行。

我们可以把 WorkBuddy 当成一个不需要睡觉、不需要领工资、且 24 小时在线的远程员工。我们关机了，它在云端继续跑；我们下班了，它还在帮我们修 Bug。这才是一只“虾”顶一个团队的实感。

WorkBuddy 还内置了国内主流的大模型，可以直接选择，再也不用费力配置 API Key 了。

 

02. 腾讯小龙虾🦞能干啥？

 

场景一：给运营小姐姐做一个“抢红包”小工具

我们运营组想搞个活动，需要一个随机抽奖的 H5。我直接在微信上发送指令：“做一个随机抽奖的 H5 页面，界面要喜庆一点，点一下按钮出现抽奖结果。”

WorkBuddy 吭哧吭哧生成完页面后，还直接提供了“一键部署”按钮~，我直接在手机上点了一下，一个抽奖的 H5 页面就出来了，还可以直接通过链接分享给朋友们。

（这里得插一句，现阶段腾讯云给了免费资源，轻量服务器和云开发资源都不要钱，相当于从写代码到上线全程零成本，这个羊毛确实可以薅。）

场景二：Vibe Coding 自己的 App

你有没有过这种经历？

脑子里突然蹦出一个点子，想赶紧做个 demo 验证一下。结果刚打开编辑器，就开始陷入无止境的“搭框架、写接口、调样式、配环境……”一整套下来，激情早没了。

现在不一样了。

我直接拿起手机，对着 WorkBuddy 说：

“用 Python+FastAPI+SQLite，给我搞一个个人记账 App，要带登录、流水记录、月报图表，最后直接部署到我 VPS 上，给个能访问的链接。”

然后，WorkBuddy 就真给我干出来了。

自己建项目结构、写代码、跑迁移、做测试、打 Docker 镜像、部署上服务器、连Git都帮我 commit 好了。

有社区老哥亲测，一晚上时间，用它从 0 到 1 搞了个小工具，不仅代码写完、部署上线，连 App Store 的多语言更新日志都自动填好了，直接提交审核。

场景三：文件管理与维护

文件管理是很多打工人都刚需的功能。尤其是做视频自媒体的伙伴，图片和视频等等各类素材又多又乱，找素材很花视角。

我这样告诉 WorkBuddy:

我需要你帮我整理这台电脑上的图片和视频：

1、先帮我扫描这台电脑上的所有图片和视频

2、帮我按照地点-时间-内容，三级目录进行整理

3、地点根据图片的说明来写，精确到市/区-县-乡镇

4、内容需要用视觉模型先进性识别，在用2-6个字总结内容

5、图片和视频的命名格式也是地点-时间-内容

WorkBuddy 很贴心的一点是能直观看到进度。

这在 OpenClaw 上都没有。有时候发一个任务半天没反应，又不知道进度到哪里了，挺闹心的，还得自己去查日志。

3分钟，WorkBuddy 处理了 1095 个文件！这个整理速度，比我手动整理快了 100 倍不止。

我们一起看看整理的结果，所有的文件都按照地点、时间分类清晰，不过 WorkBuddy 并没有识别到图片内容。

 

03. 免费“羊毛”与福利

 

我知道大家看完这种新品，第一反应可能是：“看着挺好，但会不会很贵？”或者“是不是只有程序员才能玩？”

最近确实是窗口期，有几个实打实的权益值得你花3分钟看一下：

1、新老用户直接送5000 Credits（3月31日截止）

只要是国内版用户，点进活动页面，直接就送5000积分。这5000积分足够像我刚才那样跑好几个小项目了。

领取地址：https://www.codebuddy.cn/profile/usage

2、免费云资源，开发到部署一站式（3月31日截止）

这可能是 WorkBuddy 区别于其他 AI 编程工具最大的点。别家只帮你写代码，服务器你还得自己买。

CodeBuddy 因为背靠腾讯云，直接送轻量云服务器和云开发资源。对于想折腾点小项目但不想花钱买域名买服务器的新手来说，这个太友好了。

国内版领取：https://www.codebuddy.cn/promotion/

3、百万 Credits 悬赏：发内容就能拿（4月10日截止）

重点来了！如果你觉得这玩意儿好玩，或者想挑战一下高难度，WorkBuddy 正在搞一个 “百万 Credits 悬赏” 活动。

只要你在社交平台带话题 #WorkBuddyClaw实战，发视频或图文，就能拿奖励。最高能 48,000 Credits，相当于差不多 2 年的订阅价值。

如果你本身就在研究 AI 工具，这其实是个“双重白嫖”的机会——既免费体验了新工具，还能靠产出内容赚取未来的使用额度。

 

04. 一些分享

 

WorkBuddy 肯定还不完美。

比如在手机端操作时，如果任务逻辑特别复杂（比如涉及几十个文件的联动修改），它的响应速度会比电脑端慢一些，毕竟是云端在跑。另外，虽然说是全员适用，但如果要调试非常深度的代码 Bug，还是需要有一点基础认知去阅读它给出的日志。

但瑕不掩瑜，WorkBuddy Claw 这种去设备化的思路，确实戳中了我这种不想被电脑绑死的人。

以前我们总说“等 AI 时代到来”，现在感觉是“AI时代已经在手机里了”。

如果你是开发者，可以试试让它帮你写文档、修Bug；如果你是产品/运营，可以试试让它直接给你出原型、做落地页。

写完这篇文章，我把链接发给了那位在咖啡厅改Bug的朋友。他回了我一句：“以后是不是躺沙滩上也能修 Bug 了？”

我说：“不是修 Bug，是‘指挥别人’修 Bug。”

这可能就是 WorkBuddy Claw 想做的事：让 AI 成为你的团队，而你只需要专注于思考。
