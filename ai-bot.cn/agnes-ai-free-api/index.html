Agnes AI 免费 API 实测 – 文本/图像/视频模型全免费开放

AI教程1个月前更新 AI小集
0 3

前两天凌晨，我刷到一条消息，人直接精神了。

一家全球排名第9的AI Lab，悄悄把自己三个核心模型全免费了。

文本、图片、视频，全都免费。没跟你玩虚的，不是什么阉割版、限频版，注册就能用。

这是什么操作？这风格不太熟悉呢，一看果然不是咱们国内的。这个来自新加坡的团队，还是个狠角色啊，三个模型一起发力，还都能拿到国际榜单背书。

最近网上讨论度也很高。

今天这篇，K姐就通过实测看看是不是真实免费，真的可以用。

 

01. 这个 AI 行业全球新玩家，开始冒头了

 

老读者都知道，K姐看AI有个习惯：官宣文案从来不信，Demo再炫也不上头。就盯两样东西，榜单排名和盲评成绩。

Agnes这次进的三个榜单，恰好全是测“干活能力”的。

先聊文本模型 Agnes-2.0-Flash，进的榜叫Claw-Eval。

Claw-Eval榜是开发者圈子里公认的Agent实战体检报告。Agnes-2.0-Flash在部分任务里把Gemini Flash和MiniMax M2.7都给超了。一个免费模型能在这个榜上跟付费主力模型正面刚，含金量你自己品。

再看图像模型 Agnes-Image-2.0-Flash，进的榜叫Artificial Analysis图像编辑盲评榜。

就是所有生成的图不许标是谁家模型出的，打分的人纯靠肉眼判断。品牌光环、先入为主，在这里全部失灵。

Agnes硬是挤进了这个榜的前排。友友们想想，图像编辑这条赛道上全是只做图像的老牌模型，你一个同时搞文本和视频的公司跑过来占个前排，多少有点不讲武德。

最后是视频模型 Agnes-Video-2.0，进了Artificial Analysis的Image-to-Video（带音频）排行榜。

图生视频不难，但带音频、音画同步能做好，门槛就不一样了。静态图片扔进去，出来的是有自然运动、有环境音效的动态短视频。对于需要批量产素材的团队，这个能力搭上免费API，性价比直接拉满。

文本、图像、视频三条线各自登榜，互相印证。单科爆发可以靠运气，三科全进只能是实力，没什么说的了。

 

02. 一手实测

 

榜单看完，K姐手痒。光说不练假把式，我直接把三个模型各挑了一些最考验基本功的场景，帮友友们实测一遍。

文本模型Agnes 2.0 Flash

AI营销出海工具网站

现在AI出海工具很火爆，我做了一个带3D效果的网站。视觉上粒子地球带连线自转，科技感拉满。而且不是那种干巴巴的静态网页，交互按钮能点击、弹窗有入场动画、数字会跳动递增、卡片悬停有光效。结构上也很完整，从首屏、功能、流程、定价到CTA，引导用户一步步走。

AI营销内容生成器

这个Case的亮点在于，它把AI生成内容这个概念做成了真实的交互体验，你输入产品信息、选品类选风格，点一下Generate，就能看到文案生成出来的过程。

最直观的就是那几张3D翻转卡片，正面看产品，翻过去就是AI写的文案，这个交互确实非常直观了。

Agent产品的调研分析报告

这个Case把一份17个产品的深度调研变成了一份精美的交互式演示。我直接从搜索、整理、分析到输出完整HTML幻灯片，全程没有外部工具依赖。

报告从市场总览→产品分析→排名对比→定价策略→能力矩阵→趋势洞察→出海机会，层层递进，逻辑清晰。Agnes 2.0 Flash结构化表达挺厉害的。

图片模型Agnes image 2.1 Flash

微缩世界 × 奇幻建筑

这个case选的是“露珠上的玻璃城堡”，这个创意可以把模型能做到的材质复杂度直接拉满了。

友友们看，画面里至少得有三种透明材质同时交互，露珠是水的折射，城堡是玻璃的透光和反射，水晶塔楼还有内部发光。

这三种东西搁在一起，光线从露珠穿进去再折射到城堡上，再反射到周围，物理上是非常复杂的。这就是在考验模型对真实物理世界材质的理解深度。

赛博霓虹城市夜景

适合做科技类内容的头图、视频封面,视觉冲击力强又能一眼看出模型功力。

一张图同时压三个最难的点。霓虹是多光源，模型得让每块灯牌的光正确打到墙面、伞面、风衣上，光色还要相互混合，光影一糊全盘皆崩。

最狠的是霓虹灯牌上的中文，弯曲发光的字体最容易写错笔画或糊成光斑,能把“未来已来”“AI 之城”写清楚还带发光质感的，文字渲染基本顶级。

亚马逊电商主图

亚马逊主图是商业图的标准考题，规则硬、容错低，跑得好不好一眼可辨。

提示词：纯白背景,一台深灰色无线耳机充电盒,盒盖打开,产品居中,商业棚拍,高清细节。

白底的纯净度很见功力，能把背景压到真正的 255 纯白、产品边缘还利落不发虚的,说明模型对“棚拍”这个商业语境有准确理解，耳机盒的磨砂外壳和内部金属触点应当呈现不同反光，质感拉得开的模型，材质理解力就站在第一梯队。

这一组干净利落地过关，说明模型扛得住电商带货的真实需求。

视频模型 Agnes Video v2.0

先生成一段5秒的短视频，镜头从右向左缓慢平移，画面内容自然运动，配上夜晚东京城市街道的环境音效。

镜头平移流畅，没有画面撕裂和跳帧。搭配免费API的BUFF，批量生产的成本可以直接忽略。

再看一个更偏电影化的镜头：整体氛围非常宏伟。外星人乘坐飞船在火星着陆，外星人从飞船里出来，主角缓缓前行，漫天黄沙随风飘散，很有大片感。

下面这个case考验人物表现力：一位年轻女性站在窗边，微笑转头看向镜头，镜头缓缓推近。含泪说着“我养你啊”。

美国70年代的电影画面风格，一对男女在舞会上踩着乐曲跳探戈，舞会上其他人在围观鼓掌。画面里每个人表情和动作、神态都是在场景里的。

最后，再看一下宫崎骏动画风格的影片效果，一个小女孩蹲在一只流浪狗面前说，我带你回家吧。小狗哼哼唧唧地点头。

整套 case 最妙的地方在于覆盖面，从零成本的空镜素材，到科幻大片，再从情感表演，到复古群像，再到动画风。刚好把视频模型最值钱的五种用途各验证了一遍。

无论是做带货、做短剧、做科普还是做风格化内容都能顶得上。

一个免费的视频模型能跑出这种效果，含金量摆在这里了。

 

03. 一些分享

 

聊完榜单，跑完case，K姐说点心里话。

这一年多，模型能力蹭蹭涨，API价格也蹭蹭涨。Agent成了主流方向，但天生就是Token消耗大户，多步骤、多工具、反复调用，每一次都是真金白银在烧。K姐自己跑复杂任务，用付费模型的时候点运行之前真的要做一下心理建设。

这么搞下去，能大规模玩Agent的，永远只剩那几家预算充裕的大厂。中小开发者、独立创作者、还在上学的友友们，连试错的资格都快被剥夺了。

Agnes这次免费开放，K姐看到的不是营销，是一个态度：AI这桌牌，不该只让买得起筹码的人上桌。

一个两人的小团队，接入Agnes之后，模型能力和百人大厂站在同一条起跑线。脑子里的创意可以立刻跑起来验证，“试错成本”这四个字，暂时从你的字典里删掉。

K姐自己已经把日常实验的默认模型切过去了，那种不用再盯着账单喘气的自由感，真的太爽了。

Agnes三款模型免费API已全面开放，友友们可以直接去官网薅羊毛～
