SenseNova-Skills

商汤科技开源的模块化 AI 办公技能库

标签：插件与Skills
访问官网

SenseNova-Skills是什么

SenseNova-Skills 是商汤科技开源的模块化 AI 办公技能库，专为 Agent 运行时设计。将信息图生成、PPT 制作、Excel 数据分析与深度研究四大核心办公能力拆解为独立、可复用的技能模块，遵循 agentskills.io 规范，可直接集成到 OpenClaw、Hermes Agent 等主流智能体框架中。底层依托 SenseNova 6.7 Flash-Lite 与 U1 多模态模型分工协作，通过 VLM 质量闭环与流式大文件处理技术，实现从”给出分析思路”到”自动跑完工作流、直接交付成品”的能力。

SenseNova-Skills的主要功能

  • 信息图生成：支持 87 种布局与 66 种风格，通过 VLM 自动质检与多轮迭代，将复杂数据、长报告或商业洞察转化为高密度可视化信息图。
  • PPT 生成：提供标准模式（HTML 分页 + VLM 评审 → PPTX 导出）与创意模式（每页生成 16:9 U1 全图 PNG），支持解析 PDF/DOCX/MD/TXT 等格式后自动分发任务。
  • Excel 数据分析：支持多表读取、大文件（≥1 万行）自动转 Parquet、数据清洗、条件筛选与跨表聚合；十万行级大表采用流式处理避免内存溢出；支持图片表格 OCR 识别与图表数据提取。
  • 深度研究：遵循「规划 → 多维度定向取证 → 交叉核验 → 综合判断」标准化路径，自动辨析统计口径分歧与行业观点矛盾，输出带丰富图表、可溯源的可视化报告，支持断点续跑。
  • 多源搜索：覆盖学术（ArXiv、PubMed）、开发者（GitHub、HuggingFace）、中文社交（B站、知乎、抖音）及英文社交（Reddit、X、YouTube）等垂直搜索能力。
  • 模块化集成：每个技能独立封装为目录 + SKILL.md，既可独立调用，也能组合成端到端复杂工作流，直接接入 OpenClaw、Hermes Agent 等 Agent 框架。

SenseNova-Skills的项目地址

  • 项目地址：https://github.com/OpenSenseNova/SenseNova-Skills

如何使用SenseNova-Skills

  • Agent 自动安装（推荐）：在已部署的 Agent（如 OpenClaw、Hermes Agent）中发送自然语言指令，例如”请从 https://github.com/OpenSenseNova/SenseNova-Skills 安装 SenseNova-Skills 到技能目录”，Agent 会自动克隆仓库并完成配置，安装后可能需要手动重启 Agent 服务。
  • 手动安装：通过 git clone https://github.com/OpenSenseNova/SenseNova-Skills.git --depth=1 克隆仓库，将 skills/ 目录下的子文件夹复制到对应 Agent 的技能目录（如 OpenClaw 的 ~/.openclaw/skills/ 或 Hermes 的 ~/.hermes/skills/），并根据 INSTALL.md 配置 API Key 与依赖。
  • 办公小浣熊（零配置体验）：直接访问 办公小浣熊 官网使用智能办公助手，无需安装、配置环境或 API Key，免费体验全套 SenseNova-Skills 办公能力。
  • 配置 API Key：手动安装后需在 .env 文件中配置 SenseNova 开放平台 API Key（在 https://platform.sensenova.cn/console/keys 申请免费额度），部分搜索类技能还需配置各平台的独立 API Key。
  • 端到端工作流组合：单个技能可独立调用，也可按需求串联组合，例如将数据分析结果输入深度研究，再将研究结论交给 PPT 生成技能，形成”数据 → 洞察 → 报告”的完整工作流。

SenseNova-Skills的核心优势

  • 模块化与可组合：每个技能独立封装为目录 + SKILL.md，遵循 agentskills.io 规范，既可独立调用，也能灵活串联成端到端复杂工作流。
  • 端到端成品交付：不只是输出分析建议，而是自动跑完数据清洗、图表生成、报告撰写、PPT 合成等全流程，直接交付可开会、可汇报的成品。
  • VLM 质量闭环：信息图与 PPT 生成过程中引入视觉语言模型自动质检与多轮迭代，确保布局、配色、图文匹配度达到出版级标准。
  • 大文件流式处理：针对 Excel 大文件（≥1 万行）采用流式读取 + 自动转 Parquet 技术，十万行级数据表也能稳定处理，避免内存溢出。
  • 原生多模态架构：底层依托 SenseNova 6.7 Flash-Lite 与 U1 模型分工协作——U1 负责图像生成，6.7 负责规划、扩写、质检与文字生成，实现图文深度协同。
  • 标准化深度研究路径：遵循「规划 → 多维度定向取证 → 交叉核验 → 综合判断」方法论，自动辨析统计口径分歧与观点矛盾，支持断点续跑。

SenseNova-Skills的应用场景

  • 企业绩效与 HR 数据分析：批量处理跨月份、多岗位的绩效考核 Excel 表格，自动清洗上万条数据，生成包含趋势分析、岗位对比、员工画像和改进建议的完整 Word 报告与 HTML 可视化页面，直接用于管理例会决策。
  • 行业深度调研与投资研判：一句话调研需求，自动完成多源搜索、证据交叉核验、统计口径辨析，输出带十余张图表、可溯源的专业分析报告，适用于投资人、行业研究员快速摸清市场现状。
  • 商务演示文稿自动化：根据主题、受众和风格要求，自动规划大纲、分配每页素材、生成语义化 HTML 并合成 PPTX，适用于产品发布、项目汇报、培训课件等长文本演示场景。
  • 复杂信息可视化：将长报告、商业洞察或技术概念压缩为高密度结构化信息图，支持 87 种布局与 66 种风格，适用于营销物料、知识科普、数据新闻、简历视觉设计等场景。
  • 端到端数据研究链：从原始 CSV/Excel 数据出发，先完成清洗与趋势分析，再针对异常数据触发深度研究（如”内存价格上涨原因”），最终将数据结论与研究洞察合成为一页页专业 PPT，实现”数据 → 洞察 → 报告”全链路自动化
  • 学术与竞品情报搜集：覆盖 ArXiv、PubMed、GitHub、HuggingFace、知乎、Reddit 等多源垂直搜索，自动整合碎片化资讯为结构化研判结论，适用于科研人员、产品经理、战略分析团队的日常情报工作。
