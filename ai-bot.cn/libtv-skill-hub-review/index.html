LibTV Skill Hub 实测 – 从剧本到成片一键搞定

AI教程1分钟前发布 AI小集
0 2

过去一年，Video Agent 几乎成为了 AI 视频行业最热门的方向。

很过 Agent 都能一句话生成视频片段，但当我们真正开始创作完整的视频作品时，还是绕不开生成器、画布和节点，AI 视频的核心还是写提示词和抽卡。

最近，LibTV 上线了 Skill Hub，把短剧、广告创意、MV等经验，封装成了 100+ 个可以直接调用的专业 Skill。

我们只需要一个故事剧本，十分钟就能生成一集短剧，不需要自己手搓提示词。

这质量，效果有点超预期。

今天，咱们一起体验一下 LibTV 的 AI 导演。

 

01. 100+ 专业 AI 导演随叫随到

 

我们打开 LibTV 官网，在输入框下方可以看到 Skill Hub 入口。

LibTV 邀请了 200+ 位创作者共同打造 100+ 个专业 Skill，将专业导演、自媒体、广告创意、MV 策划、短剧制作等经验封装成可调用的 Agent 工作流。

我们可以像挑选导演一样选择 Skill。比如韦斯安德森电影美学 Skill，可以帮我们深度还原韦氏标志性视听语言，帮我们设计对称构图、复古配色和舞台感的电影短片或广告，支持 15 秒到 5 分钟。

我们点击添加 Skill，输入一段带对话的广告故事：

提示词：类型：香水剧情广告

核心文案：无需邀请，自成主场。

0—8秒｜临时被取消

复古豪华酒店前台，画面采用严格的中心对称构图。

女主穿着浅色礼服，站在柜台前。

前台经理翻看登记簿，语气客气却疏离：

“林小姐，今晚的开场环节临时调整了。”

女主问：

“什么意思？”

经理合上登记簿：

“主办方安排了其他人，您不用出场了。”

女主停顿一秒：

“可我收到的是到场确认。”

经理：

“抱歉，我们也是刚刚收到通知。”

8—13秒｜短暂失落

女主独自走进酒店化妆间。

镜中的她妆容完整，却明显失去了刚才的神采。

画外音：“要不算了吧，能来已经不错了。”

她摘下耳环，又重新戴回去。

13—19秒｜香水出现

女主低头打开手包。

特写：

玻璃香水瓶、金色瓶盖、瓶中晃动的浅琥珀色液体。

她取出香水，看着镜中的自己。

没有补妆，也没有落泪。

她在耳后和手腕各喷一下。

喷雾落下的一刻，环境声全部消失。

镜中的女主缓缓抬眼。

19—25秒｜状态转变

香雾在灯光下形成一层极淡的香槟金光。

她穿上外套，合上香水瓶。

“咔哒”一声，音乐进入。

酒店电梯门在她面前打开。

她走进电梯，转身直视镜头。

电梯门关闭。

25—34秒｜进入主场

宴会厅大门打开。

女主沿着中央通道走进会场。

两侧宾客依次停止交谈，循着香气回头。

一位侍者忘记继续倒酒。

刚才取代她出场的嘉宾也看向门口。

摄影师调整镜头，对准女主。

人群自然向两边让开。

女主没有看任何人，只保持稳定的步伐向前。

34—38秒｜拿回出场权

主办方迎上来，压低声音：

“林小姐，今晚的安排已经变了。”

女主停下，却没有转头：

“安排可以变。”

她抬眼看向会场中央：

“但我的出现，不由别人决定。”

38—42秒｜产品定格

女主从镜头前走过，身后的香槟金色香气缓慢铺满画面。

切换至香水瓶定妆镜头。

字幕：

“无需邀请，自成主场。”

香水品牌“Koi”

LibTV 会通过 3 个问题，跟我们进一步确认视频细节。

确认后，LibTV 继续生成了角色卡描述，给了 4 套造型方案供我挑选。

说实话，这批生成的质量确实很不错，但是没有我特别喜欢的。

我们可以在画布中直接用大白话告诉 LibTV，重新生成一批。LibTV 会自己修改生成提示词、在画布中创建节点并重新连好工作流。

这一批的造型设计挺符合我的预期的，我们选择图一。

LibTV 开始进一步生成人物三视图和故事中提到的各个场景，每个场景都生成了 4 份供我选择，

这组场景对中心对称构图和复古色彩的处理很到位。

我们逐个确认后，LibTV 会自动整理人物、场景和产品资产，不需要我们手动连线。

LibTV 开始自动设计分镜脚本，把我们的故事拆解成一个个具体的画面。

如果觉得镜头的细节设计不够好，我们可以直接通过对话调整。

所有片段生成完成后，LibTV 还主动合成视频并且询问字幕要求，我们配上中英双语字幕。

一起看看最终生成的效果：

每一幕画面都是非常对称的韦氏构图，画面质量拉满了。

不用自己写提示词，大白话沟通就能做出这个效果，我真是太满意了。

在 Skill Hub 里，我还发现了一个宝藏女频短剧 Skill。

我们把网文或者短剧素材传上来，LibTV 可以自动创建剧本、脚本、一键出片。现在爆火的甜宠、逆袭、虐恋题材都适合。

我们来一段小说体验一下。

提示词：鸩酒灼穿喉咙时，沈昭月听见亲妹妹沈清瑶在耳边轻笑。

“姐姐，你还真以为陛下爱过你？他娶你，哄了你整整十年，为的就是让沈家心甘情愿交出兵权。如今兵权已经到手，你和那个孩子，自然也就没用了。”

殿外风雪呼啸。

父亲与兄长含冤赴死，六岁的儿子死在她怀里，嘴角还沾着尚未擦净的黑血。

沈昭月挣扎着望向殿门。

可直到视线模糊，那个与她相伴十年的男人也没有出现。

沈清瑶捏住她的下巴，将最后半杯毒酒灌进她嘴里。

“陛下说了，等你一死，便封我为后。”

剧痛吞没意识前，沈昭月只听见天牢外传来厮杀声。有人一刀劈开铁门，在漫天箭雨中喊她的名字。

那声音冷厉又急切。

是七皇子萧景珩。

可他来迟了。

再睁眼时，耳边响起丝竹声。

沈昭月跪在金殿中央，高座上的皇帝含笑看着她，诸位皇子分列两旁。

“昭月，你父亲为国镇守北境，劳苦功高。今日朕准你从诸位皇子中择一人为婿。”

满殿目光落在她身上。

太子萧承煜端坐在诸皇子席首，一袭月白锦袍，眉目温润，与前世别无二致。

察觉她望来，他缓缓搁下酒盏，隔着满殿灯火朝她微微颔首。那笑意从容而笃定，仿佛这场赐婚早有定数，她终究会选他。

前世的她就是这样走向他。

十年筹谋，她替他拉拢朝臣，为他献出母族兵权，陪他踏着尸骨登上帝位。最后换来的，是父兄惨死，幼子中毒，沈氏满门被扣上谋逆之罪。

沈昭月指尖发冷，耳边仿佛又响起幼子断断续续的哭声。

“母妃，我疼……”

她闭了闭眼。

再睁开时，她越过萧承煜，望向大殿最不起眼的角落。

“臣女谢陛下成全。臣女愿嫁七皇子萧景珩。”

角落里，一直冷眼饮酒的男人缓缓抬眸。

前世沈家覆灭，满朝避之不及，只有他带兵闯过天牢，可惜迟了一步。

萧承煜手中的玉杯裂开细纹。

满殿鸦雀无声。

七皇子生母早逝，常年领兵在外，性情冷硬，传闻他在北境一夜斩杀敌军三百，连军中的老将都不敢同他对视。

皇帝皱眉：“昭月，你可想清楚了？”

“臣女想得很清楚。”

萧承煜终于开口：“七弟久居军中，不懂怜香惜玉。昭月自幼娇贵，只怕……”

“太子殿下。”

沈昭月转头看他，唇边带着淡淡笑意。

“如今是臣女择婿。”

萧承煜的话停在唇边。

萧景珩放下酒盏，起身走到她身旁。他身量极高，玄色衣袍还带着未散的寒气，右手虎口有一道陈年刀疤。

他没有谢恩，先低头问她：“沈姑娘，当真愿意？”

前世天牢里，她已经咽不下任何东西。恍惚之间，一只染血的手抱起她，男人的声音沙哑得几乎听不清。

“沈昭月，我来晚了。”

她记得那只手上的刀疤。

“我愿意。”

LibTV 会先跟我们确认视觉风格和画面尺寸。

LibTV 对题材定位，核心冲突、爽点分析特别全面。并且还会分析爆款元素。

看看 LibTV 写的人物小传，主角表面上是什么身份、实际是什么身份，角色外貌如何、性格如何，分析的特别细，就像导演给演员讲戏一样。

主角是什么身份，外貌特真如何，标志性动作、性格特点如何，LibTV 在“开拍”前都讲得透透的。

我只给了 400 字左右第一集的剧情，LibTV 按照剧情思路自动帮我补全了接下来的内容，构建了一个完整的短剧大纲。

继续生成了第一集的完整剧本：

最精彩的是，LibTV 会根据对白的字数来规划片段的时长，对视频节奏的把握非常棒。

随后，LibTV 在画布中创建了四个角色资产、场景资产和道具资产。在画布中添加了相应节点。

沈清瑶的角色设定图不符合古装剧的风格，我们也可以直接大白话让它重新生成。

LibTV 生成了 6 个不带配音的画面，我们通过对话让 LibTV 继续生成环境音和人声配音，并且在合适的地方加上古风 BGM 配乐。

LibTV 会基于分镜立马生成匹配的旁白、音乐，自动合成成片。

我们点击视频合成节点：

因为人声配音和画面是分开生成的，有些画面和声音不能完全对上，我们可以在这个剪辑页面手动微微调整一下。

导出就是开头的小短剧了，整体故事逻辑很顺，视频片段之间的衔接非常自然。

LibTV 还支持故事版和节点式工作流双视图切换。我们可以精细控制每一个镜头、节点和运镜方式。

爆款一键拉片

复刻日常学习做视频，少不了拆解爆款、拉片逐帧学习。LibTV 上有一个爆款拉片复刻 Skill，可以帮我们一键复刻同款。

在画布中选择skill，添加参考视频和需要替换的内容。我上传了大话西游里比较经典的一个片段。

提示词：将@参考视频 中的女生替换成@女角色 的脸部、发型和服装，脸部必须是@女角色 女生的脸，将男生替换成@男角色 的脸部、发型和服装，其他都不变。

LibTV 会先确认需求，视频比例和清晰度要求。然后开始对原视频进行拉片分析。

先理解视频中的镜头节奏、叙事结构和素材安排，按照我们的要求生成全新的脚本。

我们只需要点点确认，就能复刻全部视频内容。画面结构、运镜方式、人物情绪都十分还原原片。

 

我们看到任何喜欢的爆款，都可以上传参考，让 Agent 拆解、重组并生成自己的成片版本。

 

02. 1000w 激励计划

 

LibTV Agent 投入 1000 万元创作激励，扶持优质 Skill 创作者，共同打造全球最大的专业视频 Skill Hub。

如果我们本身已经有比较稳定的创作风格或者流程，可以在 LibTV 中通过对话生成 Skill 或者直接创建。

每一位发布 Skill 的创作者都有机会获得官方流量、现金激励、品牌共创及未来长期收益。

 

03. 一些分享

 

会不会写提示词，会不会操作工具，不再是 AI 视频的门槛。

过去，一套成熟的创作方法可能只掌握在专业编导手中。普通人就算是使用同样顶尖的视频生成模型，面对空白的画布依然不知道该如何起笔。

Skill Hub 把专业创作者的审美偏好、提示词、工作流程和镜头感，打包封装成可以直接调用的 Skill。每一个有想法、有故事的普通人，也可以通过复用公开 Skill，站在专业导演的肩膀上，去创作出高质量的作品。

对于专业创作者来说，这也是一个新的机会。导演、广告创意、短剧团队和自媒体创作者，可以把自己反复验证过的方法做成 Skill。

以前大多数创作者只能靠接项目出售时间，未来也许可以通过出售和授权成熟工作流的方式，让自己的专业经验与审美偏好在云端被无限次地调用和复利变现。
